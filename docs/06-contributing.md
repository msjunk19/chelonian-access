# Contributing

Thank you for your interest in contributing to **Chelonian Access**.

This project is focused on building a reliable, modular **vehicle access control system**, so contributions should prioritize stability, clarity, and maintainability.

---

## Contribution Guidelines

### Core Principles

All contributions should follow these principles:

- **Vehicle access first** – do not break core unlock functionality  
- **Keep it simple** – avoid unnecessary complexity  
- **Modular design** – keep features isolated and reusable  
- **Non-blocking logic** – avoid `delay()` where possible  
- **Testable code** – ensure logic can be tested without hardware
